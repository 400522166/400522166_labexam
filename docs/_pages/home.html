Introduction

Lone Wolf Theme is infamous for abstracting away the UI code and help us focus only on the content and the functionalities.

As an embedded systems software developer I often struggle with the UI for my personal website which hosts my work portfolio.

Have used a freely available website theme for jekyll, like minimal mistakes. But it didn't meet all my expectations besides I wanted to make minimal changes to the UI and make major changes to the functionalities like:

  • HTML-2-PDF conversion for blog posts
  • Blog posts reading progress bar.
  • Custom Animations.
  • Idea submission form

So I decided to make my own theme which is simple, abstract, elegant and efficient.

It is aimed at the solo freelancers who host their portfolios, take ideas form submission, have a todo lists to achieve their goals and host a blog to pen their thoughts.
